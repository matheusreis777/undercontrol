# Under Control

## Visão Geral

O **Under Control** é uma landing page institucional e portfólio voltado para apresentar soluções de gestão empresarial.

O objetivo do projeto é servir como uma vitrine moderna e profissional, apresentando:

- a empresa e seu propósito
- os serviços oferecidos
- o portfólio de produtos e soluções digitais
- tecnologias utilizadas no desenvolvimento

---

# Objetivos do Projeto

O objetivo do **Under Control** é centralizar a apresentação de diferentes soluções digitais em um único portal.

Principais objetivos:

- Servir como portfólio de soluções
- Apresentar uma identidade visual moderna e profissional
- Facilitar o contato e a visualização de serviços
- Manter uma vitrine atualizada de produtos

---

# Estrutura do Projeto

O projeto é desenvolvido como uma aplicação web moderna utilizando **Next.js**.

### Tecnologias Utilizadas

- **Next.js**: Framework para React com foco em performance e SEO.
- **Tailwind CSS**: Estilização moderna e responsiva.
- **Framer Motion**: Animações suaves e interativas.
- **Lucide React**: Ícones profissionais.

---

# Landing Page / Portfólio

O **Under Control Web** funciona como o portal principal.

Ele tem como objetivo apresentar:

- **Home**: Apresentação principal com foco em controle e eficiência.
- **Sobre**: Propósito da empresa e visão de mercado.
- **Produtos**: Demonstração visual de sistemas desenvolvidos (ex: Gestão de Ar-Condicionado).
- **Soluções**: Detalhamento técnico e diferenciais competitivos.
